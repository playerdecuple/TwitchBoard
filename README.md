# TwitchBoard | 트위치보드
> 시청자가 단합력을 보여 직접 해보는 게임!

## 소개(Introduce)
`깨고싶어 너와함께`라는 컨텐츠를 보고 바로 생각해낸 미니 유틸리티입니다!  
채팅창에서 메시지들을 불러와 가장 많이 투표된 키가 입력됩니다!

## 설치(Installation)

### 필요한 것 (Required)
[Java 8](https://java.com/)

### 개발자 (Developer)
1. 클론합니다. ```git clone https://github.com/playerdecuple/TwitchBoard```
2. IntelliJ IDEA에서 엽니다. 끝!
